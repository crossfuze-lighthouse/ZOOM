How scoring works
Reference
1. User completion percentage

Each user is scored across 7 checklist steps. Completion is calculated as completed steps divided by 7, then converted to a percentage.

Example: 3 of 7 complete = 43% · 5 of 7 complete = 71% · 7 of 7 complete = 100%
2. Overall completion

Overall completion is the average user completion score across all tracked users.

3. Live users

A user counts as live when these are complete: licence allocated, user activated, profile/settings complete, and signals/intent basics complete.

4. Regional progress

Regional progress is the average completion score for users within that region only, such as UK or USA.

5. Suggested user status bands
0% = Not started
1–49% = Getting started
50–84% = In progress
85–99% = Nearly ready
100% = Complete
Checklist steps used for scoring
1. Licence allocated
2. User activated
3. Profile / settings complete
4. Signals / intent basics complete
5. Target account list built
6. Target account alert created
7. First outreach launched
